This workspace was updated with an image preview modal and download button.

Files changed:
- index.html: added an Image preview section and modal markup
- assets/site.css: added styles for the modal and zoomable images
- assets/site.js: added JS to open/close modal and set download link

How to use:
- Click any image with class `zoomable` to open the viewer.
- Click the download button or right-click the image and choose "Save image as...".

If you want multiple images, duplicate the `<img class="zoomable" src="...">` element in the section.
